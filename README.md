# Login_page
Login Page Project
About This Project

This is a simple Login Page created using HTML and CSS.

The page is divided into two sections:

Left side contains an image of a washing machine.
Right side contains a login form where users can enter their email and password.
Features
Responsive two-column layout using Flexbox.
Email input field.
Password input field.
Forgot Password option.
Login button.
Register link for new users.
Clean and modern user interface.
Technologies Used
HTML5
CSS3
What I Learned

While building this project, I learned:

How to create forms using HTML.
How to use Flexbox for page layouts.
How to style input fields and buttons using CSS.
How to align content horizontally and vertically.
How to connect labels with input fields using the for and id attributes.
How to create a clean login page design.
